Este site é um projeto interdisciplinar da Etec, envolvendo as disciplinas Estudos Avançados em Ciências da Natureza e suas Tecnologias, Estudos Avançados em Matemática e suas Tecnologias, Programação Web I, Análise e Projeto de Sistemas, Língua Portuguesa, Banco de Dados I e Laboratório de Investigação Científica. Foi desenvolvido com o objetivo disseminar o conhecimento à respeito de medicamentos para as pessoas, principalmente, os idosos.

As tecnologias utilizadas para o desenvolvimento desse site foram: linguagens HTML5, CSS3, Javascript e o framework Bootstrap.
